DevFest Toulouse 2016 - Kotlin for Android

Develop a reactive Android app with Kotlin - Animé par Arnaud G.

November 15, 2016 Julien Zanon

3 minute read

Présentation Cette première présentation de la journée est une présentation technique. Arnaud G. nous présente le language Kotlin comme une alternative tout à fait viable pour le développement d’applications Android. Kotlin v1.0 est marqué “Production Ready” par son éditeur JetBrains. Il requiert au minimum Java 6 pour le faire tourner. Soit dit en passant, aujourd’hui, QUI aurait l’idée de se lancer dans un nouveau développement sans utiliser la dernière version de Java disponible?!
